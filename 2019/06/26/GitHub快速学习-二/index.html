创建仓库

  • New repository
    • repository name 仓库名
    • description 仓库说明
    • Public Or Private 仓库是否公开
    • Initialize this repository with a README 自动初始化仓库并且设置README文件
    • Add .gitignore 这个设定帮助我们把不需要在Git仓库中进行版本管理的文件记录在 .gitignore文件中
    • Add a license 添加许可协议
  • Create repository 完成仓库创建

README.md
